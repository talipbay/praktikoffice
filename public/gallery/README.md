# Gallery Images

This folder contains images for the gallery section organized by categories.

## Folder Structure

- `offices/` - Office space images (office-1.jpg to office-6.jpg)
- `meeting/` - Meeting room images (meeting-1.jpg to meeting-6.jpg)
- `coworking/` - Coworking space images (coworking-1.jpg to coworking-6.jpg)
- `lounge/` - Lounge area images (lounge-1.jpg to lounge-6.jpg)
- `fitness/` - Fitness zone images (fitness-1.jpg to fitness-6.jpg)

## Image Requirements

- Format: JPG or PNG
- Aspect ratio: 4:3 (recommended)
- Resolution: 1200x900px or higher
- File naming: [category]-[number].jpg (e.g., office-1.jpg, meeting-2.jpg)

Each category should have exactly 6 images for the 3x2 grid layout.
